<div>

<div> tagu izmanto, lai definētu sadaļu no HTML dokumenta. To izmanto kā konteineri HTML elementiem, kuriem pēc tam piešķir stilu, izmantojot CSS vai JavaScript.

<span>

<span> tags ir līdzīgs <div> tagam - tas izveido konteineri HTML elementiem. Galvenā atšķirība ir tā, ka <span> definē konteineri rindā, bet <div> definē blokveida konteineri.

<style>


<style> izmanto, lai definētu stila informāciju (CSS) HTML dokumentā. Tajā definē to, kā lapai vajadzētu pārlūkprogrammā parādīties.
